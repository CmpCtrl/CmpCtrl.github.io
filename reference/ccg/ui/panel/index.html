text_to_scripts 

text_to_scripts(filename: Path) -> dict

Returns a _scripts dictionary for ReactiveHTML based on a string

Args: filename (str): The js script

Returns: Dict: The output _scripts

Example:

txt=''' ... render=()=>{ ... console.log(data) ... } ... value=()=>{ ... my_func(value) ... } ... ''' text_to_scripts(txt) {'render': 'console.log(data)', 'value': 'my_func(value)'}

Note: The script must be in the following format: Based on MarcSkovMarksen's suggestions for ReactiveHTML Scripts.
